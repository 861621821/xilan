### 注意事项  
本组件基于element-ui开发，所以你的项目需要先安装[element-ui](https://element.eleme.cn/#/zh-CN/component/installation)

### 开始使用
``` js
import xilan from 'xilan'
import 'xilan/dist/xl-ui.css'

Vue.use(xilan)
```
### 组件列表
+ xl-table  
    >支持行内编辑
+ xl-filter
    >Popover形式条件过滤组件
+ xl-toolTip
    >文本溢出打点，hover时弹气泡显示
